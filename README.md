## How to run the tests
1. Clone the repository and enter its directory:

```bash
git clone https://github.com/Afeka-DevTools/26b-10142-unittesting-itay-and-yuval.git
cd 26b-10142-unittesting-itay-and-yuval
```

2. Run all tests (Unix / macOS):

```bash
./gradlew test
```

3. Run all tests (Windows):

```powershell
gradlew.bat test
```

4. Run only the app module tests:

```bash
./gradlew :app:test
```

5. View the test report after running tests:

- JUnit report: `app/build/reports/tests/test/index.html`
